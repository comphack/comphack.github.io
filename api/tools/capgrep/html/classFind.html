Detailed Description

Dialog used to find a packet.

A packet can be found with three different methods: command code, text, binary. Command codes are input as hex values (like in the filter dialog). Text can be converted to one of multiple encodings selectable from a drop-down list. Once the text is converted, the binary sequence is then searched for. Binary searches convert a hex sequence into a byte encoded sequence to match in the packet data. The results of a search are displayed in a list and the search term for the list is indicated. Double-clicking an item in the list displays the matching packet/command in the main window. Note that the search results have the sample command code filter as the list in the main window.
