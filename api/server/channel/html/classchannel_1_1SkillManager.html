Activate the skill of a character or demon.

The client will have the charge information sent to it if there is a charge time, otherwise it will execute automatically.

Parameters
sourcePointer of the entity that activated the skill

activationObjectIDID of the object used as the activation target

ctxSpecial execution state for the skill (ex: free cast, counter)
Returns
true if the skill was activated successfully, false otherwise
