Binary data is part of the client game data. It consists of the files in the BinaryData directory. This directory consists of two sub-directories: Client and Shield. Files in the Client directory are not encrypted; howeever, files in the Shield directory are. Encrypted files have the file extension sbin or stxt. Files that start with the prefix C are client related files. Files that start with the prefix S are server related files. Files that have no prefix are shared by both the client and server. This is a commonly accepted convention, not a rule. Most file include a format text file with the postfix Format that describe the structure of the file.
