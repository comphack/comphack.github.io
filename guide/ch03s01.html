Overview

Figure 3.1. High level interaction between many of the main COMP_hack components.


The project contains many components, some of which are Internet facing. A Client who wishes to play the game will usually interact with the Web Server, Lobby Server, and Channel Server (in that order). A Game Master, Server Administrator, or Developer may also interact with the World Server and ChanMan Server, as well as place a Logger between their Game Client and a Channel Server to observe and debug. Each of these components will be discussed in greater detail.

Updater

The Updater is a replacement for the original updater provided by CAVE. The new updater provides the following enhanced features:

  • A time stamp check to allow for fast loading when there are no new updates.

  • Compression of the file listing (hashlist.dat file) for faster updates.

  • An embedded web client to prevent caching of update files and mismatch errors.

  • The ability to select more than one lobby server to connect to when starting the Game Client.

Browser

The Browser is used by the player and game masters to interact with the Web Server to create and manage account information and access the item store. The Browser is not a component of this project.
