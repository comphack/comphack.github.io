Shortcuts

Figure 4.1. Example of a shortcut bar that contains the currently summoned partner's skills, blank entries, a command, skills, items, and an equip.


The shortcut bar contains 10 pages of 24 shortcuts which are usually laid out in 2 rows. A shortcut can be for a command (emotes, sit, gift partner, etc.), skill, demon skill, item (consumable or equipment), menu, or partner (contracted devil). Items will show if they are equipped or how many of the item are in the inventory (or both in the case of bullets). The first 8 shortcuts on the bar are reserved to display the currently summoned partner's skills. That leaves 16 customizable shortcuts per page for a total of 160 shortcuts.

Each shortcut has 9 bytes of data. The first value is an 8-bit integer that indicates the type of shortcut. Possible types values can be seen in Table 4.1, “Shortcut Types”. The remaining 8 bytes are dependent on the shortcut type.

Table 4.1. Shortcut Types

IDDescription
0Blank
1Command
2Skill
3Devil
4Item
5Equip
8Menu
9Fusion Spell


Blank Shortcuts

Blank shortcuts always have 8 bytes of zero.
